# EventPlannerApp

EventPlannerApp is a Python-based application designed to streamline event management. It includes features such as analytics dashboards, event tracking, and more.

## Features
- **Analytics Dashboard**: View statistics on upcoming events, completed events, revenue, and attendees.
- **Revenue Trends**: Visualize monthly revenue trends.
- **Event Management**: Manage upcoming and past events efficiently.

## Requirements
To run this application locally, you need:
- Python 3.10 or above
- Required Python libraries (install via `pip install -r requirements.txt`)
